# Tobator Agents

This file provides context for AI agents to understand and interact with the Tobator project.

## Monorepo Structure

This project is structured as a monorepo. The main components are:

-   `api`: A Python API built with FastAPI.

## api

-   **Framework**: FastAPI
-   **Entrypoint**: `api/src/api/main.py`
-   **Dependencies**:
    - `api/requirements.in` (direct dependencies, e.g. `fastapi`, `pytest`, `ruff`)
    - `api/requirements.txt` (autogenerated by [pip-tools](https://github.com/jazzband/pip-tools), fully pinned)
-   **Tests**: `api/tests/`
-   **Environment Management**: `direnv` and `venv`
-   **Linting**: `ruff`
-   **Testing**: `pytest` (configured in `pyproject.toml` to find the `api` package)

### Managing dependencies

To add or remove a dependency:

1. Edit `requirements.in` and add or remove the package name (e.g., `fastapi`, `pytest`, `ruff`, etc.).
2. Run the following command in the `api` directory to update `requirements.txt`:

   ```bash
   pip-compile --upgrade
   ```

**Do not edit `requirements.txt` directly.** It is autogenerated and should only be updated via `pip-compile`.

### How to run the API

1.  `cd api`
2.  `direnv allow` (may happen automatically, will also install dependencies)
3.  `uvicorn api.main:app --reload --app-dir src`

### How to run tests

1.  `cd api`
2.  `direnv allow` (this will also install dependencies)
3.  `pytest`
