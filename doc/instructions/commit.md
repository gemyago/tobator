# Instruction to commit changes

* You will be given a list of files to commit.
* **Note** If not explicitly mentioned, commit all updated files (`git add .`).
* When committing, make sure to provide a sensible message. Figure out the message from chat history. 
* If no relevant chat history is available, figure out the message based on the files changed (using `git diff`)
* Do not ask for confirmation, you're **APPROVED** to commit.
